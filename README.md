# tripplanner

관광지와 행사 정보를 탐색하고 여행 일정을 구성하는 React 웹 프로젝트입니다. 화면 흐름, API 연동, 로그인 상태에 따른 기능 제어를 중심으로 작업했습니다.

## 주요 기능

- 카테고리별 관광지 검색
- 방문자 리뷰 확인
- 여행 일정 생성 및 관리
- 회원가입, 로그인, 세션 확인
- Spring Boot API 연동

## 담당 범위

- React 컴포넌트 구조 설계
- 사용자 흐름 기반 화면 구현
- 로그인 여부에 따른 화면과 기능 제어
- API 요청과 응답 처리
- Figma 기반 UI 협업

## 기술

- React
- JavaScript
- CSS
- React Router

## 실행

```bash
npm install
npm start
```
