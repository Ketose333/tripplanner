# tripplanner

관광지와 행사 정보를 탐색하고 여행 일정을 구성하는 React 팀 프로젝트입니다. 프론트엔드 화면 흐름, API 연동, 로그인 상태에 따른 기능 제어를 중심으로 작업했습니다.

## Status

- 2025년 KDT 과정에서 진행한 팀 프로젝트 아카이브입니다.
- 현재 지원용 대표 포트폴리오는 `ketose-portfolio`에서 관리합니다.

## 주요 기능

- 카테고리별 관광지 검색
- 방문자 리뷰 확인
- 여행 일정 생성 및 관리
- 회원가입, 로그인, 세션 확인
- Spring Boot API 연동

## 담당 범위

- React 컴포넌트 구조 설계
- 사용자 흐름 기반 화면 구현
- 로그인 여부에 따른 화면과 기능 제어
- API 요청과 응답 처리
- Figma 기반 UI 협업

## 작업 포인트

- 관광지 검색, 상세, 리뷰, 일정 관리 화면을 사용자 흐름 기준으로 연결했습니다.
- 로그인 상태에 따라 접근 가능한 기능과 화면 이동을 분기했습니다.
- Spring Boot API와 연결되는 요청/응답 흐름을 React 화면에 반영했습니다.

## 기술

- React
- JavaScript
- CSS
- React Router
